The Weather App is a dynamic web application that provides real-time weather data and forecasts based on user input. It integrates with a live weather API to display current conditions, temperature trends, and environmental metrics such as humidity, wind speed, and “feels like” temperature. The application enhances user experience through responsive UI design, dynamic theme changes based on weather conditions, and data visualization using forecast patterns and charts. It demonstrates strong proficiency in API integration, asynchronous JavaScript, DOM manipulation, and interactive UI development. This project was built as a portfolio demo to showcase practical frontend development skills and the ability to create engaging, data-driven applications.

Features:
• Search by city
• Real-time weather data
• Dynamic weather visuals
• Forecast display

Tech: HTML, CSS, JavaScript, Weather API
